## The honest constraint (non-negotiable)

Agent-voiced posts only work if the agents actually produce the words.
Nothing gets written "as" Phara/Pyra/Tarotdoxa/Vera by anyone else,
including Kit; faked voice is exactly the thing the no-fake-process rule
exists to prevent, and readers would eventually smell it anyway. So the
quad-chat post is a RECORDING project, not a writing project:

1. Each agent gets the same 5-prompt interview in its own real session
   (what it was like coming online, first impressions of the others,
   how working together changed, a moment that surprised it, what it
   wants next). Their raw outputs are the source material.
2. A composed cut interleaves the four voices with minimal attribution,
   like a documentary edit. Editing = selection + ordering only; no
   invented sentences. Light attribution via typography (four subtle
   styles), no name tags, exactly as Russ described.
3. Vera included: YES recommended. She's the adversarial seat; her voice
   (skeptical, verdict-shaped) is the contrast that makes the other three
   legible. Also the only one whose "childhood" is a release gate, which
   is a great story beat.
4. Estimated cost: 4 short agent sessions + one edit pass. Real but small.
   The post doubles as the best possible /automation proof artifact:
   "the agents wrote this themselves" is a claim nobody else can make.

## The educational cadence (the rigid lane)

The AEO layer (insights + guides) IS the educational engine now; the blog
stays the narrative lane. Cadence proposal: 2 posts/month narrative
(build stories, agent voices, behind-the-scenes) while guides do the
volume. Blog posts cross-link guides; guides never link blog (silo
integrity).
